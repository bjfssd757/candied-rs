# Candied-rs

---

Candied - Rust библиотека, предоставляющая макросы для добавления больше синтаксического сахара и уменьшения количества кода

## Оглавление
* [Функционал](#функционал)
* [Документация](#документация)

## Функционал

* Генерация векторов в диапазонах и с условиями для каждых элементов
* Макрос для вызова функции для всех указанных элементов (с возвращением результата и без)
* Switch-case конструкция
* Макрос, аналогичный тернарному оператору
* Макрос, вызывающий блок кода во время выхода из области видимости функции или блока кода, где он был вызван
